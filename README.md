# 🍅 ポモドーロ × ゴール

ポモドーロタイマーと目標管理で集中力を最大化するPWAアプリ。

## 機能

- **ポモドーロタイマー**: 25分集中 / 5分小休憩 / 15分大休憩
- **目標管理**: 日々の目標を設定・追跡
- **統計**: 日・週・月の集中時間を可視化
- **オフライン対応**: PWAでインストール可能
- **通知**: ポモドーロ完了時にブラウザ通知

## 技術スタック

React + TypeScript + Vite + vite-plugin-pwa

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```
